# A mirror of ywvim script

ywvim 是一个 vim 里的五笔输入法。

已经删除掉了不必要的中文标点。

使用`Ctrl-Space`或`Ctrl-\`或`Ctrl-@`来触发。

带有五笔的码表。

## Install

    git clone https://github.com/scateu/ywvim ~/.vim/pack/plugins/start/ywvim
